@Target(allowedTargets = [AnnotationTarget.CLASS])
annotation class StateGenie(rootName: String, isParcelable: Boolean)

StateGenie triggers our implementation of SymbolProcessor from Kotlin Symbol Processor(KSP) where we validate the Symbols.

Validation Check includes:

is `resolvable` &&
is `KSClassDeclaration` &&
is `classKind == ClassKind.INTERFACE` &&
is modifiers does not contains `Modifier.FUN`.
Content copied to clipboard

After validation each symbols are processed by an implementation of KSVisitor provided by KSP to create the sealed interface for the respective symbols.

Following source code:

@StateGenie(
  rootName = "NewUiSate",
  isParcelable = true
)
interface UiState{
  @ToState(
    stateName = "Success"
  )
  val data: String

  @ToState(
    stateName = "Loading"
  )
  val loading: Unit

  @ToState(
    stateName = "Error"
  )
  val error: Throwable
}
